## Getting Started

Simply open `index.html` in your web browser to start using the application.

### Usage

1. Fill in the shift details:
   - Lab Name
   - Staff Name
   - Date
   - Time Slot (Morning, Afternoon, Evening, or Night)

2. Click "Add Shift" to create a new shift

3. View all scheduled shifts in the "Scheduled Shifts" section

4. Delete shifts using the "Delete Shift" button on each shift card
