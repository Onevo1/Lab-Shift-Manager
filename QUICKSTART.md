# Quick Start Guide - Lab Shift Manager

## üöÄ Getting Your Team's Link

After the repository owner enables GitHub Pages, your team will access the shift manager at:

## üì± First Time Setup

### For the Team Administrator:

1. **Access the application** using the link above
2. **Add team members**:
   - Enter each team member's name in the "Team Members" section
   - Click "Add Member"
   - Repeat for all team members

3. **Create initial shifts**:
   - Select a team member from the dropdown
   - Choose the shift date
   - Set start and end times (e.g., 09:00 to 17:00)
   - Add any notes (optional) like "Equipment maintenance" or "Training session"
   - Click "Add Shift"

4. **Export the schedule**:
   - Click "Export Data" button
   - Save the JSON file
   - Share this file with your team members

### For Team Members:

1. **Access the application** using the team link
2. **Import the schedule** (if provided by admin):
   - Click "Import Data"
   - Select the JSON file shared by your administrator
   - Confirm the import

## üí° Daily Usage

### Viewing Your Schedule
- Open the application link
- Your shifts appear in the "Schedule" section
- Use the date filter to view specific days
- Click "Show All" to see the complete schedule

### Adding a Shift
1. Select your name from the dropdown
2. Pick the date
3. Enter start and end times
4. Add any relevant notes
5. Click "Add Shift"

### Managing Your Data
- **Backup**: Click "Export Data" weekly to save your schedule
- **Restore**: Use "Import Data" to restore from a backup
- **Clear**: "Clear All Data" removes everything (use carefully!)

## ‚ö†Ô∏è Important Notes

### Data Storage
- Your data is stored **locally on your device**
- Each person has their own copy of the schedule
- Data is **not automatically shared** between team members

### For Shared Schedules
Since data isn't automatically synced:

**Option 1: Single Manager**
- One person maintains the master schedule
- They export and share updates weekly/monthly

**Option 2: Individual Use**
- Each person manages their own shifts
- Use for personal schedule tracking

**Option 3: Manual Sync**
- Team members export their schedules
- Admin imports and merges them
- Redistributes the combined schedule

## üÜò Troubleshooting

### "I lost my schedule!"
- Import your last exported backup file
- If no backup exists, the data cannot be recovered

### "The link doesn't work"
- Check that GitHub Pages is enabled (ask repository owner)
- Wait 5-10 minutes after initial setup
- Try clearing browser cache

### "My shifts disappeared"
- Don't use incognito/private mode (data won't persist)
- Don't clear browser data/cookies
- Make regular backups using Export

### "Changes aren't saving"
- Check browser console for errors (F12)
- Ensure you're not in private browsing mode
- Try a different browser
